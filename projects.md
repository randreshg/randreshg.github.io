# Projects

## APXM

[A-PXM](https://github.com/randreshg/apxm) is a parallel execution model for agentic AI. The project spans graph IR design, compiler lowering, runtime scheduling, and backend integration for tool- and model-driven workflows.

## AgentMate

[AgentMate](https://github.com/randreshg/agentmate) is a Rust framework for building AI agents and CLI applications with tools, streaming, memory, and APXM-backed execution.

## Tully

[Tully](https://github.com/randreshg/tully) is a local-first, domain-neutral experiment tracking and research context system for reproducible technical workflows.

## CARTS Benchmarks

[carts-benchmarks](https://github.com/randreshg/carts-benchmarks) contains benchmarks and harnesses for evaluating CARTS and related execution-contract experiments.
